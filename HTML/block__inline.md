HTML은 element의 type에 따라 default display value 존재  

2가지 Type 존재
- Block type
- Inline type 

## Block-level Elements
---
- __항상 새로운 new line에서 시작됨__
- 전체 width를 사용함
- top/bottom margin 존재

#### block level elements
```
<section>
<div>
<header>
<footer>
<main>
<table>
<ul>
<ol>
<p>
<li>
...
```

## Inline Elements
---
- new line으로 시작하지 않음
- top/bottom margin 없음

#### Inline elements
```
<a>
<i>
<img>
<span>
<map>
<script>
<br>
...
```
